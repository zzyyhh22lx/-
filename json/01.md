<h1 align="center">JSON</h1>



#### JSON形式

```json
{
    "name": "zhangsan",
    "age": 12,
    "lover": [
        "coding",
        "swimming"
    ]
}
```





**JSON数据的属性名字key与属性值value必须使用双引号""包裹**

```javascript
let a1 = '{"name":"听风是风","age":"26"}';// √
let b1 = "{'name':'听风是风','age':'26'}";// X


        var tag = '{"key":"1"}';
		//JSON.parse 将符合json形式的字符串转换为对象
        tag = JSON.parse(tag);
        console.log(tag.key);


		//JSON.stringify()
		//把json形式的对象转换为字符串
```
