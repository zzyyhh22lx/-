# vue-li打包

```shell
# 下载淘宝cnpm镜像（ 速度快 ）
npm install -g cnpm --registry=https://registry.npmmirror.com

# 最新稳定版
cnpm install vue 

# 全局安装 vue-cli
cnpm install --global vue-cli

# 创建一个基于 webpack 模板的新项目
vue init webpack my-project
# 一直按enter

# 进入项目，安装并运行
cd my-project
cnpm install
cnpm run dev

# DONE  Compiled successfully in 4388ms
# > Listening at http://localhost:8080

# 项目打包
npm run build

```
