# 异步



```javascript
function f() {
    setTimeout(function() {
        var data = 10;
        return data;
    }, 100);
    //进行到这里直接返回了，不会等到回调函数内部return再返回
}

fn()//undefined


// 回调函数，获得 异步操作的 数据
// 把函数作为参数就是为了获取函数内部的异步操作结果
function fn(callback) {
    setTimeout(function() {
        var data = 10;
        callback(data);
    }, 100);
}

fn(function(data) {
    console.log(data);
})

// promise
function func() {
    new Promise(function(resolve, reject) {
        setTimeout(function() {
            var data = 10;
            resolve(data);
        }, 100);
    }).then(function(data) {
        console.log(data);
    })
}

func();
```



异步：

- `setTimeout`

- `readFile`
- `writeFile`
- `ajax`



### 处理异步的数据，最好使用回调函数

===> 可以封装函数，传递函数作为参数执行不同的操作。
