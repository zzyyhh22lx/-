#### fs.readFile

获得二进制数据或者字符串

用toString()方法，或者将readFile设置第二个参数`'utf8'`设置识别码，

#### 路由设计

| -------- | ---------------- | ------- | -------------------------- | ------------ |
| GET      | /students        |         |                            | 渲染首页     |
