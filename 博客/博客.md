### 路由设计

| 路径      | 方法 | get参数 | post参数                  | 权限( 登录) | 备注         |
| --------- | ---- | ------- | ------------------------- | ----------- | ------------ |
| /         | GET  |         |                           |             | 渲染首页     |
| /register | GET  |         |                           |             | 渲染页面     |
| /register | POST |         | email、username、password |             | 处理注册页面 |
| /sign     | GET  |         |                           |             | 渲染登录页面 |
| /sign     | POST |         | email、password           |             | 处理登录请求 |
| /signout  | GET  |         |                           |             | 处理退出请求 |
|           |      |         |                           |             |              |
